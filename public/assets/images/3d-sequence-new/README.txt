Folder Lokasi Foto Beruntun (3D Sequence) Baru
==============================================

Folder ini disiapkan khusus untuk menaruh berkas foto beruntun (sekuensial) untuk kebutuhan visualisasi 3D tema baru di halaman website.

Panduan Pengunggahan:
1. Letakkan seluruh berkas gambar sekuensial Anda langsung di dalam folder ini:
   Path: public/assets/images/3d-sequence-new/

2. Format penamaan file disarankan menggunakan indeks angka berurutan yang konsisten (misal: 000.jpg, 001.jpg, 002.jpg, ... dst) agar mudah diparsing dan diposisikan secara dinamis menggunakan Javascript/Canvas di halaman utama web.
